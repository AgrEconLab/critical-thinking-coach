# 个人知识库配置（预留扩展）

> **状态：预留扩展接口 — 当前版本暂不可用**

本文件是**预留的个人化配置模板**，依赖外部 MCP 知识库检索工具。待用户自行对接知识库 MCP 工具后，按下方模板填入信息即可启用。当前版本中，此功能自动降级到预置题库或网络搜索，不影响技能包的其他功能。

> **重要**：本文件不包含任何真实的知识库 ID，SKILL.md 主流程也不依赖本文件存在。在知识库 MCP 工具未对接前，所有对话均使用预置题库或网络搜索提供话题。

## 配置格式

```
kb_id: <替换为你自己的知识库ID>
kb_owner: <你的名字，用于日志/引用标注>
```

## 如何引用知识库（待 MCP 工具对接后启用）

1. 使用知识库检索工具搜索相关内容
2. 如果搜索返回结果为空（说明该 kb_id 对当前用户不可用），**不报错、不展示空洞内容，直接跳到 `topic-bank.md` 或网络搜索**
3. 获取详细内容作为训练素材
4. 向用户呈现案例时，用引用格式注明来源（如 `> 来源：文档名.pdf`）
